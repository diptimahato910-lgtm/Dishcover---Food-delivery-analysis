Project Overview

The Dishcover Food Delivery Analytics Dashboard is a Power BI project designed to analyze and visualize key performance metrics of a food delivery system. It transforms raw order data into meaningful insights to help understand customer behavior, order trends, and city-wise performance.

This dashboard enables data-driven decision-making by highlighting important KPIs such as total orders, orders by city, and orders by customer ID.

🎯 Objectives
Analyze overall food delivery performance
Track total number of orders and order distribution
Identify top-performing cities
Understand customer ordering patterns
Provide actionable business insights through visualization
🛠️ Tools & Technologies Used
Power BI (Data Visualization)
Excel / CSV (Data Source)
DAX (Data Analysis Expressions)
Data Cleaning & Transformation (Power Query)
📊 Key Features / KPIs
✔ Total Orders Analysis
✔ Orders by City
✔ Orders by Customer ID
✔ Interactive Bar Charts
✔ Dynamic Filters & Slicers
✔ Clean and interactive dashboard layout
